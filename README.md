# Natural Language Processing (NLP)

This repository contains a series of 11 classes on Natural Language Processing.

It is part of the first cohort of the Higher Education Technology Degree in Data Science at Fatec Jundiaí.

Each class will be published weekly (every Thursday), starting on April 2, 2026.

---

## Course Content

- Introduction to Natural Language Processing  
- Python Review (NumPy) and Introduction to PyTorch  
- Text Preprocessing  
- Tokenization Techniques  
- Vectorization and Embeddings  
- Classical NLP Models  
- Neural Networks for NLP  
- Attention Mechanism  
- Transformers Architecture  
- Large Language Models (LLMs)  
- Applications of NLP
